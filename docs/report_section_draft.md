# 보고서 3. 탐구 내용 - 2) 탐구 활동 내용 초안

본 탐구에서는 확률과 조건부확률의 개념이 실제 인공지능 예측 모델에서 어떻게 활용되는지 알아보기 위해, 나이브 베이즈 분류기를 이용한 한국어 스미싱 문자 탐지 프로그램을 제작하였다. 기존의 단순한 예측 예시에서 나아가, 실제 한국어 문자 메시지 데이터셋을 활용하여 문자가 일반 문자인지, 피싱 또는 스미싱 의심 문자인지를 분류하는 프로그램을 구현하였다.

프로그램은 문자 메시지의 내용을 입력값으로 사용하고, 각 문자를 `일반 문자`와 `피싱/스미싱 문자`의 두 범주로 나누어 학습한다. 문자에 포함된 단어들은 단어 빈도 형태로 변환되며, 나이브 베이즈 분류기는 각 단어가 특정 범주에서 등장할 조건부확률을 학습한다. 예를 들어 `계좌`, `본인인증`, `링크`, `정지`, `배송 오류`와 같은 표현이 피싱 또는 스미싱 문자에서 자주 등장한다면, 이러한 단어가 포함된 새 문자 역시 스미싱일 가능성이 높게 계산된다.

이때 핵심이 되는 수학 개념은 조건부확률과 베이즈 정리이다. 어떤 문자에 여러 단어가 포함되어 있을 때, 프로그램은 `P(일반 문자 | 단어들)`과 `P(스미싱 문자 | 단어들)`을 각각 계산하여 더 큰 값을 갖는 범주를 최종 예측 결과로 선택한다. 나이브 베이즈 분류기는 여러 단어의 등장 여부가 서로 독립이라고 가정하기 때문에, 각 단어의 조건부확률을 곱하는 방식으로 전체 확률 점수를 계산할 수 있다.

또한 학습 데이터에서 한 번도 등장하지 않은 단어 때문에 전체 확률이 0이 되는 문제를 해결하기 위해 라플라스 스무딩을 적용하였다. 라플라스 스무딩은 각 단어가 최소한 한 번은 등장한 것처럼 보정하여, 새로운 문자가 들어왔을 때도 안정적으로 확률을 계산할 수 있게 해준다.

프로그램 제작 과정은 데이터셋 불러오기, 문자 데이터 전처리, 학습용 데이터와 테스트용 데이터 분리, 단어 빈도 벡터화, 나이브 베이즈 모델 학습, 테스트 데이터 평가, 직접 입력한 문자 예측의 순서로 진행하였다. 이를 통해 확률은 단순히 사건의 가능성을 계산하는 개념에 그치지 않고, 실제 인공지능 모델이 데이터를 바탕으로 판단을 내리는 과정에 직접 활용될 수 있음을 확인하였다.
