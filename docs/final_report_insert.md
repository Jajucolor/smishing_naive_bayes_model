# 최종 보고서에 넣을 수 있는 통합 문단

## 3. 탐구 내용 - 2) 탐구 활동 내용

본 탐구에서는 확률과 조건부확률의 개념이 실제 인공지능 예측 모델에서 어떻게 활용되는지 알아보기 위해, 나이브 베이즈 분류기를 이용한 한국어 스미싱 문자 탐지 프로그램을 제작하였다. 프로그램은 한국어 문자 메시지를 입력받아 해당 문자가 일반 문자인지, 피싱 또는 스미싱 의심 문자인지를 확률적으로 분류한다.

사용한 데이터는 한국어 문자 메시지와 라벨로 이루어진 데이터셋이며, 문자 내용은 `content`, 분류 라벨은 `class` 열을 이용하였다. 라벨은 일반 문자와 피싱/스미싱 문자로 구분하였다. 프로그램은 먼저 문자 메시지를 단어 단위로 나누고, 각 단어가 몇 번 등장하는지를 세어 단어 빈도 벡터로 변환하였다. 이후 다항 나이브 베이즈 분류기를 사용하여 각 단어가 일반 문자 또는 스미싱 문자에서 등장할 조건부확률을 학습하였다.

수학적으로는 어떤 문자에 여러 단어가 포함되어 있을 때, `P(일반 문자 | 단어들)`과 `P(스미싱 문자 | 단어들)`을 각각 계산하여 더 큰 값을 갖는 범주를 최종 예측 결과로 선택한다. 나이브 베이즈 분류기는 여러 단어가 서로 독립적으로 등장한다고 가정하므로, 각 단어의 조건부확률을 곱하여 전체 확률 점수를 계산할 수 있다. 또한 학습 데이터에서 한 번도 등장하지 않은 단어 때문에 확률이 0이 되는 문제를 막기 위해 라플라스 스무딩을 적용하였다.

프로그램 제작은 데이터셋 불러오기, 데이터 전처리, 학습용 데이터와 테스트용 데이터 분리, 단어 빈도 벡터화, 나이브 베이즈 모델 학습, 성능 평가, 직접 입력한 문자 예측 순서로 진행하였다. 학습이 끝난 뒤에는 테스트 데이터를 이용하여 모델의 정확도와 혼동행렬을 확인하였다. 혼동행렬을 통해 실제 일반 문자를 일반 문자로 맞힌 경우, 실제 스미싱 문자를 스미싱 문자로 맞힌 경우, 그리고 두 종류의 오분류 사례를 구분하여 분석하였다.

또한 모델이 어떤 단어를 근거로 판단하는지 확인하기 위해 분류별 주요 단어를 분석하였다. 특정 단어가 스미싱 문자에서 자주 등장하고 일반 문자에서는 적게 등장한다면, 그 단어가 포함된 문자는 스미싱일 가능성이 높게 계산된다. 이를 통해 조건부확률이 단순한 이론적 개념이 아니라 실제 인공지능 분류 모델의 판단 근거로 사용될 수 있음을 확인하였다.

## 4. 내용 요약 및 자신의 의견에 넣을 수 있는 문장

이번 탐구를 통해 확률과 조건부확률이 인공지능 모델의 핵심적인 판단 원리로 활용될 수 있음을 알게 되었다. 특히 나이브 베이즈 분류기는 각 단어가 특정 범주에서 등장할 조건부확률을 계산하고, 이를 바탕으로 새로운 데이터의 범주를 예측한다는 점에서 확률과 통계 과목과 직접적으로 연결되었다.

또한 실제 한국어 문자 데이터셋을 사용하여 스미싱 탐지 프로그램을 구현하면서, 수학 개념이 정보보안 문제 해결에도 활용될 수 있음을 확인하였다. 다만 나이브 베이즈 분류기는 단어들이 서로 독립이라고 가정하기 때문에 문맥을 완전히 이해하지는 못한다. 따라서 더 정교한 탐지 시스템을 만들기 위해서는 문장의 의미와 순서를 함께 고려하는 모델로 확장할 필요가 있다고 생각하였다.
